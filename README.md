# Projek 2 Mata Kuliah Berpikir Komputasional (Sistem Peminjaman Buku di Perpustakaan)
Program ini merupakan sistem perpustakaan sederhana yang memungkinkan pengguna mencari buku, meminjam, dan mengembalikannya. 
Seluruh proses seperti pengecekan stok, pencatatan peminjaman, perhitungan denda, hingga pembayaran ditangani otomatis oleh program, dengan data disimpan dan diperbarui melalui file JSON.

## File Source Code adalah mainprogram.py dan program ini hanya dapat berjalan dengan menggunakan database dengan file Database_buku.json dan database_peminjambuku.json

## Fitur yang tersedia : 
1. Mencari Buku Berdasarkan Genre dan Judul
2. Melakukan peminjaman buku
3. Melakukan pengembalian buku
4. Melakukan pengecekan status peminjaman Buku
5. Mengganti tanggal
6. Pembayaran Denda
